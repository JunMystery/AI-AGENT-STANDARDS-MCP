# Rules Distill

Use this skill to turn repeated workflow guidance into durable standards.

## What to Apply

- Collect evidence from multiple skills, docs, PR reviews, or recurring task failures.
- Promote only cross-cutting rules that apply beyond one narrow feature or stack.
- Update the smallest durable home: project standards, checklist, prompt template, or agent instruction.
- Avoid duplicating the same rule across many files unless agent discovery requires it.
- Record assumptions and open questions when the evidence is incomplete.
