# Skill Scout

Use this skill before creating new skill content.

## Trigger

- User asks whether a skill already exists
- User wants to create a new skill
- User wants to fork or extend a workflow helper

## What to Apply

- Search existing local docs first.
- Prefer an existing match over new content.
- Keep the final set small and task-specific.
